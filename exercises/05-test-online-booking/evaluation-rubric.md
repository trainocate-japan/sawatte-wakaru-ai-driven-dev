# 採点基準（5段階評価）

| 評価観点 | 説明 | 満点 |
|-----------|------|------|
| 1. テスト対象の整理 | 何をテストするかが明確である | 5 |
| 2. テスト観点の妥当性 | 正常系と異常系が整理されている | 5 |
| 3. テストコードの活用 | AIを使ってテストコードを作成できている | 5 |
| 4. 実行結果の記録 | Pass/Fail/保留が整理されている | 5 |
| 5. 修正サイクル | エラー解析と修正判断が記録されている | 5 |
| 6. 学びの整理 | AI任せにしない視点が書けている | 5 |

合計：30点満点
