# 採点基準（5段階評価）

| 評価観点 | 説明 | 満点 |
|-----------|------|------|
| 1. システム概要・機能一覧 | システムの目的と主要機能が分かりやすく整理されている | 5 |
| 2. 画面・機能仕様 | 利用者、目的、入力、表示、操作、正常時の結果が整理されている | 5 |
| 3. 業務ルール | 利用ルールや条件が明確に書かれている | 5 |
| 4. エラー仕様 | エラー条件と利用者への案内が整理されている | 5 |
| 5. AI活用と見直し | AIの出力を確認し、修正・整理した形跡がある | 5 |

合計：25点満点

## 評価の目安

### 5点
- とてもよくできている
- 初学者でも内容を説明できる
- 次の設計演習にそのままつなげやすい

### 3点
- 基本はできている
- 一部に曖昧さや不足がある
- 設計に進む前に少し追加確認が必要

### 1点
- まだ整理が足りない
- 内容が曖昧で、設計につなげにくい
- AIの出力をそのまま貼っただけになっている
