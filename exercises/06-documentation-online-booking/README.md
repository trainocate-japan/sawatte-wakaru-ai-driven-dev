# 演習06：ドキュメント作成（オンライン予約管理・AI活用）

演習04の実装結果と演習05のテスト結果をもとに、生成AIと Cursor を使って **README や利用者向けドキュメント** を作成します。  
この演習では第5章の後半に合わせて、**AIを使ってドキュメント作成を効率化する流れ** を体験します。

## ゴール
- 実装内容をもとに README のたたき台をAIに作らせられる
- 読み手を意識して、初学者向けに分かりやすい説明へ整えられる
- セットアップ手順、主な機能、制約、テスト方法を整理できる
- コードやテスト結果と矛盾しないドキュメントに修正できる

## 所要時間（約30分）
- Step1 ドキュメントの対象と読者を決める（5分）
- Step2 READMEのたたき台を生成する（10分）
- Step3 内容を見直して整える（10分）
- Step4 成果物整理と共有（5分）

## 成果物
- `deliverables-<team>.md`：ドキュメント作成結果のまとめ
- README.md または README 相当の文書
- 必要に応じて利用者向け説明や制約一覧
